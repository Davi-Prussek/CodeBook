# CodeBook

> Uma biblioteca digital para organizar e consultar conceitos de programação e estrutura de páginas HTML, estilização com CSS e conceitos JavaScript.

**Atenção**  
Este repositório está em fase de desenvolvimento e funciona como um **protótipo**.  
Algumas funcionalidades ainda não estão em funcionamento além do frontend estar em uma versão beta.

---

## Sobre o projeto

O **CodeBook** é um projeto pessoal e independente criado com o objetivo de
armazenar, organizar e consultar informações sobre:

- Tags HTML
- Propriedades CSS
- Conceitos e funções JavaScript
- Exemplos de uso
- Categorias técnicas (forms, semântica, layout, meta, etc.)

A ideia é funcionar como uma **biblioteca de referência**, voltada em sua maioria para consulta rápida no dia a dia.

---

## Tecnologias utilizadas até o presente momento do projeto

### Backend
- Python
- Django
- Django REST Framework
- MySQL

### Frontend
- Vue 3
- Vue Router
- Pinia
- Axios
- Font Awesome
